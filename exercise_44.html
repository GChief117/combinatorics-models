Exercise 44 — Reverse Is Anti-Distributive

Induction proof — establishing a property of reverse over concatenation.

The Scenario

The reverse function is anti-distributive over concatenation. That is, for all sequences $s$ and $t$:

$$\text{reverse}(s \frown t) \;=\; \text{reverse}\; t \;\frown\; \text{reverse}\; s$$

Unlike ordinary distributivity ($f(a \cdot b) = f(a) \cdot f(b)$), anti-distributivity swaps the order of the operands. This is analogous to matrix transpose: $(AB)^T = B^T A^T$.

Given Definitions

Definition of reverse

reverse.1: $\;\text{reverse}\;\langle\rangle = \langle\rangle$

reverse.2: $\;\text{reverse}(\langle x \rangle \frown s) = (\text{reverse}\; s) \frown \langle x \rangle$

Reading the definition: The empty sequence reversed is still empty. The reverse of a sequence starting with $x$ followed by $s$ is the reverse of $s$ with $x$ appended at the end.

Concatenation Laws

cat.1a: $\;\langle\rangle \frown s = s$  —  empty sequence is left identity

cat.1b: $\;s \frown \langle\rangle = s$  —  empty sequence is right identity

cat.2: $\;(\langle x \rangle \frown u) \frown t = \langle x \rangle \frown (u \frown t)$  —  associativity

Induction Principle for Sequences

To prove $P(s)$ for all finite sequences $s$, it suffices to show:

  1. Base case: $P(\langle\rangle)$
  2. Inductive step: For every element $x$ and sequence $u$, if $P(u)$ then $P(\langle x \rangle \frown u)$

This works because every finite sequence is either $\langle\rangle$ or can be written as $\langle x \rangle \frown u$ for some $x$ and shorter sequence $u$.

Statement to Prove

Task: Prove by structural induction on $s$ that for all $t$: $$\text{reverse}(s \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}\; s$$

Let $P(s)$ be the predicate:

$$P(s) \;\equiv\; \forall\, t \;\bullet\; \text{reverse}(s \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}\; s$$

We prove $P(s)$ for all sequences $s$ by induction on $s$. This yields two sub-problems:

Part (a) — Base Case: $s = \langle\rangle$

Task: Show $P(\langle\rangle)$, i.e., show the base case that for all $t$:

$$\text{reverse}(\langle\rangle \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}\;\langle\rangle$$

Step 1 — Understand the task

This is the base case of our induction. We must show that the anti-distributivity equation holds when $s$ is the ⟨⟩. We do this by simplifying both sides independently using equational reasoning and showing they are equal.

Step 2 — Identify the approach

We will simplify the left-hand side using cat.1a (the left identity law), and the right-hand side using reverse.1 and cat.1b (the right identity law). Both sides should reduce to the same expression.

Step 3 — Plan the components

LHS: reverse(⟨⟩ ⌢ t) → apply cat.1a to get reverse t.

RHS: reverse t ⌢ reverse ⟨⟩ → apply reverse.1 to get reverse t ⌢ ⟨⟩ → apply cat.1b to get reverse t.

Both simplify to reverse t, so the base case holds.

We show both sides equal reverse $t$ by a chain of equalities.

Complete answer

Base case chain:

$$\text{reverse}(\langle\rangle \frown t) \;\underset{\text{cat.1a}}{=}\; \text{reverse}\; t \;\underset{\text{cat.1b}}{=}\; (\text{reverse}\; t) \frown \langle\rangle \;\underset{\text{reverse.1}}{=}\; \text{reverse}\; t \;\frown\; \text{reverse}\;\langle\rangle$$

Both sides equal $\text{reverse}\; t$. The base case $P(\langle\rangle)$ holds.

Part (b) — Inductive Step: $s = \langle x \rangle \frown u$

Task: Assume $P(u)$ holds (the inductive hypothesis), i.e., for all $t$:

$$\text{reverse}(u \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}\; u$$

Show $P(\langle x \rangle \frown u)$ holds, i.e., for all $t$:

$$\text{reverse}((\langle x \rangle \frown u) \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}(\langle x \rangle \frown u)$$

Step 1 — Understand the task

This is the inductive step. We assume the inductive hypothesis for $u$ and must prove the equation for the larger sequence $\langle x \rangle \frown u$. Our strategy is to simplify the LHS and RHS independently until they match.

Step 2 — Identify the approach

We work on each side separately:

  • LHS: Use cat.2 (associativity) to re-bracket, then reverse.2 to peel off $x$, then the inductive hypothesis.
  • RHS: Use reverse.2 on reverse$(\langle x \rangle \frown u)$, then cat.2 (associativity) to re-bracket.

Both sides will reduce to the same expression.

Step 3 — Plan the components

LHS chain:

  • reverse$((\langle x \rangle \frown u) \frown t)$ → cat.2 → reverse$(\langle x \rangle \frown (u \frown t))$ → reverse.2 → reverse$(u \frown t) \frown \langle x \rangle$ → IH → (reverse $t$ ⌢ reverse $u$) ⌢ $\langle x \rangle$

RHS chain:

  • reverse $t$ ⌢ reverse$(\langle x \rangle \frown u)$ → reverse.2 → reverse $t$ ⌢ (reverse $u$ ⌢ $\langle x \rangle$) → cat.2 → (reverse $t$ ⌢ reverse $u$) ⌢ $\langle x \rangle$

Both equal (reverse $t$ ⌢ reverse $u$) ⌢ $\langle x \rangle$.

Solution

We simplify the LHS and RHS independently until both reach the same expression.

Left-Hand Side:

Write the LHS of the goal, substituting $s = \langle x \rangle \frown u$:

$$\text{reverse}((\langle x \rangle \frown u) \frown t)$$

Build Step 2 — Apply cat.2 (associativity) to re-bracket

SymbolPlain EnglishWhat it does here
$(\langle x \rangle \frown u) \frown t = \langle x \rangle \frown (u \frown t)$cat.2 — associativity of concatenationre-brackets so that $\langle x \rangle$ is the leading singleton, enabling reverse.2

Apply cat.2 inside the argument of reverse. This moves the brackets so the leading singleton $\langle x \rangle$ is separated:

$$\text{reverse}((\langle x \rangle \frown u) \frown t) \;=\; \text{reverse}(\langle x \rangle \frown (u \frown t))$$

Build Step 3 — Apply reverse.2 to peel off $x$

SymbolPlain EnglishWhat it does here
$\text{reverse}(\langle x \rangle \frown s) = (\text{reverse}\; s) \frown \langle x \rangle$reverse.2 — reverse of a cons is the reverse of the tail with the head appendedpeels off $x$ to the end, with $s$ replaced by $u \frown t$

Apply reverse.2 with $s := u \frown t$. The leading $\langle x \rangle$ moves to the end:

$$\text{reverse}(\langle x \rangle \frown (u \frown t)) \;=\; \text{reverse}(u \frown t) \;\frown\; \langle x \rangle$$

Build Step 4 — Apply the inductive hypothesis

SymbolPlain EnglishWhat it does here
$\text{reverse}(u \frown t) = \text{reverse}\; t \frown \text{reverse}\; u$IH — the inductive hypothesis assumed for $u$replaces reverse$(u \frown t)$ with the anti-distributed form

Apply the inductive hypothesis to replace reverse$(u \frown t)$:

$$\text{reverse}(u \frown t) \;\frown\; \langle x \rangle \;=\; (\text{reverse}\; t \;\frown\; \text{reverse}\; u) \;\frown\; \langle x \rangle$$
LHS result: $(\text{reverse}\; t \;\frown\; \text{reverse}\; u) \;\frown\; \langle x \rangle$

Right-Hand Side:

Build Step 5 — Start with the RHS

Write the RHS of the goal, substituting $s = \langle x \rangle \frown u$:

$$\text{reverse}\; t \;\frown\; \text{reverse}(\langle x \rangle \frown u)$$

Build Step 6 — Apply reverse.2 to expand reverse$(\langle x \rangle \frown u)$

SymbolPlain EnglishWhat it does here
$\text{reverse}(\langle x \rangle \frown u) = (\text{reverse}\; u) \frown \langle x \rangle$reverse.2 — reverse of a cons is the reverse of the tail with the head appendedexpands reverse$(\langle x \rangle \frown u)$ into (reverse $u$) $\frown$ $\langle x \rangle$

Apply reverse.2 to the second operand:

$$\text{reverse}\; t \;\frown\; \text{reverse}(\langle x \rangle \frown u) \;=\; \text{reverse}\; t \;\frown\; (\text{reverse}\; u \;\frown\; \langle x \rangle)$$

Build Step 7 — Apply cat.2 (associativity) to re-bracket

SymbolPlain EnglishWhat it does here
$a \frown (b \frown c) = (a \frown b) \frown c$cat.2 — associativity of concatenation (used in reverse direction)re-brackets the three-way concatenation so it matches the LHS result

Apply cat.2 (associativity) to re-bracket the three-way concatenation. Here $a =$ reverse $t$, $b =$ reverse $u$, $c = \langle x \rangle$:

$$\text{reverse}\; t \;\frown\; (\text{reverse}\; u \;\frown\; \langle x \rangle) \;=\; (\text{reverse}\; t \;\frown\; \text{reverse}\; u) \;\frown\; \langle x \rangle$$
RHS result: $(\text{reverse}\; t \;\frown\; \text{reverse}\; u) \;\frown\; \langle x \rangle$

Build Step 8 — Conclude LHS = RHS

LHS (from Build Step 4) $= (\text{reverse}\; t \;\frown\; \text{reverse}\; u) \;\frown\; \langle x \rangle$

RHS (from Build Step 7) $= (\text{reverse}\; t \;\frown\; \text{reverse}\; u) \;\frown\; \langle x \rangle$

They are identical. The inductive step holds.

Complete answer

LHS chain:

$$\text{reverse}((\langle x \rangle \frown u) \frown t) \;\underset{\text{cat.2}}{=}\; \text{reverse}(\langle x \rangle \frown (u \frown t)) \;\underset{\text{rev.2}}{=}\; \text{reverse}(u \frown t) \frown \langle x \rangle \;\underset{\text{IH}}{=}\; (\text{reverse}\;t \frown \text{reverse}\;u) \frown \langle x \rangle$$

RHS chain:

$$\text{reverse}\;t \frown \text{reverse}(\langle x \rangle \frown u) \;\underset{\text{rev.2}}{=}\; \text{reverse}\;t \frown (\text{reverse}\;u \frown \langle x \rangle) \;\underset{\text{cat.2}}{=}\; (\text{reverse}\;t \frown \text{reverse}\;u) \frown \langle x \rangle$$

LHS $=$ RHS $= (\text{reverse}\;t \;\frown\; \text{reverse}\;u) \;\frown\; \langle x \rangle$. The inductive step is complete.

$$\forall\, s, t \;\bullet\; \text{reverse}(s \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}\; s$$
