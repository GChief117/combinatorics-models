Exercise 45 — Reverse of Reverse Is Identity

Induction proof — showing that reverse is an involution on sequences.

The Scenario

We prove by structural induction on sequences that for all sequences $s$:

$$\text{reverse}(\text{reverse}\; s) = s$$

That is, reverse is an involution: applying it twice gives back the original sequence. This is analogous to negation ($-(-x) = x$) or logical complement ($\neg\neg p = p$).

Given Definitions

Definition of reverse

reverse.1: $\;\text{reverse}\;\langle\rangle = \langle\rangle$

reverse.2: $\;\text{reverse}(\langle x \rangle \frown s) = (\text{reverse}\; s) \frown \langle x \rangle$

Reading the definition: The empty sequence reversed is still empty. The reverse of a sequence starting with $x$ followed by $s$ is the reverse of $s$ with $x$ appended at the end.

Concatenation Laws

cat.1a: $\;\langle\rangle \frown s = s$  —  empty sequence is left identity

cat.1b: $\;s \frown \langle\rangle = s$  —  empty sequence is right identity

Result from Exercise 44 (Anti-Distributivity)

Anti-distributive law: For all sequences $s$ and $t$:

$$\text{reverse}(s \frown t) = \text{reverse}\; t \;\frown\; \text{reverse}\; s$$

Proved in Exercise 44; used here as an established result. Unlike ordinary distributivity ($f(a \cdot b) = f(a) \cdot f(b)$), anti-distributivity swaps the order of the operands.

Induction Principle for Sequences

To prove $P(s)$ for all finite sequences $s$, it suffices to show:

  1. Base case: $P(\langle\rangle)$
  2. Inductive step: For every element $x$ and sequence $t$, if $P(t)$ then $P(\langle x \rangle \frown t)$

This works because every finite sequence is either $\langle\rangle$ or can be written as $\langle x \rangle \frown t$ for some $x$ and shorter sequence $t$.

Part (a) — Base Case: $s = \langle\rangle$

Task: Show $P(\langle\rangle)$, i.e., show the base case:

$$\text{reverse}(\text{reverse}\;\langle\rangle) = \langle\rangle$$

Step 1 — Understand the task

This is the base case of our induction. We must show that applying reverse twice to the ⟨⟩ gives back ⟨⟩. We do this by equational reasoning, simplifying from the inside out.

Step 2 — Identify the approach

The expression reverse(reverse ⟨⟩) has two nested applications of reverse. We apply reverse.1 to the inner call first, then to the outer call. Two applications of the same rule reduce the expression to ⟨⟩.

Step 3 — Plan the components

Chain: reverse(reverse ⟨⟩) → apply reverse.1 to inner → reverse ⟨⟩ → apply reverse.1 again → ⟨⟩.

The result is ⟨⟩, which equals $s$. The base case holds.

We obtain $\langle\rangle$, which equals $s$. The base case $P(\langle\rangle)$ holds.

Part (b) — Inductive Step: $s = \langle x \rangle \frown t$

Task: Assume $P(t)$ holds (the inductive hypothesis), i.e.:

$$\text{reverse}(\text{reverse}\; t) = t$$

Show $P(\langle x \rangle \frown t)$ holds for an arbitrary element $x$, i.e.:

$$\text{reverse}(\text{reverse}(\langle x \rangle \frown t)) = \langle x \rangle \frown t$$

Step 1 — Understand the task

This is the inductive step. We assume the inductive hypothesis for $t$ and must prove the equation for the larger sequence $\langle x \rangle \frown t$. Our strategy is to simplify the left-hand side reverse(reverse($\langle x \rangle$ ⌢ $t$)) step by step using equational reasoning until it equals $\langle x \rangle$ ⌢ $t$.

Step 2 — Identify the approach

We work from the inside out. First, apply reverse.2 to the inner reverse. Then the key step: apply the anti-distributive law from Exercise 44 to the outer reverse. Next, simplify reverse ⟨x⟩ down to ⟨x⟩ using cat.1b, reverse.2, reverse.1, and cat.1a. Finally, apply the inductive hypothesis to collapse reverse(reverse $t$) to $t$.

Step 3 — Plan the components

The proof chain has 7 lines. Here is the roadmap:

  1. reverse(reverse(⟨x⟩ ⌢ t)) → reverse.2 on inner → reverse((reverse t) ⌢ ⟨x⟩)
  2. → anti-distributive (Ex 44) → reverse ⟨x⟩ ⌢ reverse(reverse t)
  3. → cat.1b backwards on ⟨x⟩ → reverse(⟨x⟩ ⌢ ⟨⟩) ⌢ reverse(reverse t)
  4. → reverse.2 → ((reverse ⟨⟩) ⌢ ⟨x⟩) ⌢ reverse(reverse t)
  5. → reverse.1 → (⟨⟩ ⌢ ⟨x⟩) ⌢ reverse(reverse t)
  6. → cat.1a → ⟨x⟩ ⌢ reverse(reverse t)
  7. → IH → ⟨x⟩ ⌢ t

Solution

We simplify the LHS step by step until it equals $\langle x \rangle \frown t$.

Build Step 1 — Start with the LHS and apply reverse.2 to the inner reverse

$\text{reverse}$reverse — produces the sequence with elements in the opposite orderapplied twice: inner and outer

Write the LHS and apply reverse.2 to the inner reverse. Here reverse(⟨x⟩ ⌢ t) becomes (reverse t) ⌢ ⟨x⟩:

$$\text{reverse}(\text{reverse}(\langle x \rangle \frown t)) \;=\; \text{reverse}((\text{reverse}\; t) \frown \langle x \rangle)$$

Build Step 2 — Apply the anti-distributive law (Exercise 44)

SymbolPlain EnglishWhat it does here
$\text{reverse}(s \frown t) = \text{reverse}\; t \frown \text{reverse}\; s$anti-distributive — reversing a concatenation swaps the order of the two partssplits the outer reverse across the concatenation, swapping the two pieces

This is the key step. The expression (reverse $t$) ⌢ ⟨x⟩ does not match the pattern ⟨y⟩ ⌢ u needed by reverse.2, so we cannot apply reverse.2 directly. Instead, apply the anti-distributive law from Exercise 44, with $s :=$ reverse $t$ and $t := \langle x \rangle$:

$$\text{reverse}((\text{reverse}\; t) \frown \langle x \rangle) \;=\; \text{reverse}\;\langle x \rangle \;\frown\; \text{reverse}(\text{reverse}\; t)$$

Build Step 3 — Rewrite $\langle x \rangle$ as $\langle x \rangle \frown \langle\rangle$ using cat.1b backwards

SymbolPlain EnglishWhat it does here
$s \frown \langle\rangle = s$cat.1b — the empty sequence is the right identity for concatenationused backwards: $\langle x \rangle = \langle x \rangle \frown \langle\rangle$, allowing reverse.2 to apply

We need to simplify reverse ⟨x⟩, but ⟨x⟩ is not in the form ⟨y⟩ ⌢ $s$ that reverse.2 requires. Apply cat.1b backwards to introduce a trailing ⟨⟩:

$$\text{reverse}\;\langle x \rangle \;\frown\; \text{reverse}(\text{reverse}\; t) \;=\; \text{reverse}(\langle x \rangle \frown \langle\rangle) \;\frown\; \text{reverse}(\text{reverse}\; t)$$

Build Step 4 — Apply reverse.2 to expand reverse$(\langle x \rangle \frown \langle\rangle)$

SymbolPlain EnglishWhat it does here
$\text{reverse}(\langle x \rangle \frown s) = (\text{reverse}\; s) \frown \langle x \rangle$reverse.2 — the head moves to the end after reversing the tailwith $s := \langle\rangle$, gives (reverse $\langle\rangle$) $\frown$ $\langle x \rangle$

Now the argument matches the pattern ⟨x⟩ ⌢ $s$ with $s = \langle\rangle$. Apply reverse.2:

$$\text{reverse}(\langle x \rangle \frown \langle\rangle) \;\frown\; \text{reverse}(\text{reverse}\; t) \;=\; ((\text{reverse}\;\langle\rangle) \frown \langle x \rangle) \;\frown\; \text{reverse}(\text{reverse}\; t)$$

Build Step 5 — Apply reverse.1 to simplify reverse $\langle\rangle$

SymbolPlain EnglishWhat it does here
$\text{reverse}\;\langle\rangle = \langle\rangle$reverse.1 — reverse of the empty sequence is the empty sequencesimplifies reverse $\langle\rangle$ to $\langle\rangle$ inside the expression

Apply reverse.1 to replace reverse ⟨⟩ with ⟨⟩:

$$((\text{reverse}\;\langle\rangle) \frown \langle x \rangle) \;\frown\; \text{reverse}(\text{reverse}\; t) \;=\; (\langle\rangle \frown \langle x \rangle) \;\frown\; \text{reverse}(\text{reverse}\; t)$$

Build Step 6 — Apply cat.1a to simplify $\langle\rangle \frown \langle x \rangle$

SymbolPlain EnglishWhat it does here
$\langle\rangle \frown s = s$cat.1a — the empty sequence is the left identity for concatenationremoves the leading $\langle\rangle$, giving $\langle x \rangle$

Apply cat.1a: ⟨⟩ ⌢ ⟨x⟩ $=$ ⟨x⟩. This completes the simplification of reverse ⟨x⟩ down to ⟨x⟩:

$$(\langle\rangle \frown \langle x \rangle) \;\frown\; \text{reverse}(\text{reverse}\; t) \;=\; \langle x \rangle \;\frown\; \text{reverse}(\text{reverse}\; t)$$

Build Step 7 — Apply the inductive hypothesis

SymbolPlain EnglishWhat it does here
$\text{reverse}(\text{reverse}\; t) = t$IH — the inductive hypothesis assumed for $t$collapses the double reverse to $t$, completing the proof

Apply the inductive hypothesis to replace reverse(reverse $t$) with $t$:

$$\langle x \rangle \;\frown\; \text{reverse}(\text{reverse}\; t) \;=\; \langle x \rangle \;\frown\; t$$

This equals $\langle x \rangle \frown t$, which is exactly $s$. The inductive step holds.

Complete answer

Inductive step — full chain:

$$\text{reverse}(\text{reverse}(\langle x \rangle \frown t))$$ $$\underset{\text{reverse.2}}{=}\; \text{reverse}((\text{reverse}\; t) \frown \langle x \rangle)$$ $$\underset{\text{anti-dist. (Ex 44)}}{=}\; \text{reverse}\;\langle x \rangle \;\frown\; \text{reverse}(\text{reverse}\; t)$$ $$\underset{\text{cat.1b}}{=}\; \text{reverse}(\langle x \rangle \frown \langle\rangle) \;\frown\; \text{reverse}(\text{reverse}\; t)$$ $$\underset{\text{reverse.2}}{=}\; ((\text{reverse}\;\langle\rangle) \frown \langle x \rangle) \;\frown\; \text{reverse}(\text{reverse}\; t)$$ $$\underset{\text{reverse.1}}{=}\; (\langle\rangle \frown \langle x \rangle) \;\frown\; \text{reverse}(\text{reverse}\; t)$$ $$\underset{\text{cat.1a}}{=}\; \langle x \rangle \;\frown\; \text{reverse}(\text{reverse}\; t)$$ $$\underset{\text{IH}}{=}\; \langle x \rangle \;\frown\; t$$

So reverse(reverse($\langle x \rangle \frown t$)) $= \langle x \rangle \frown t$. The inductive step is complete.

Summary

PartWhat it provesKey laws used
(a) Base case$P(\langle\rangle)$: reverse(reverse $\langle\rangle$) $= \langle\rangle$reverse.1 (twice)
(b) Inductive step$P(t) \Rightarrow P(\langle x \rangle \frown t)$: LHS reduces to $\langle x \rangle \frown t$reverse.2, anti-distributive (Ex 44), cat.1b, reverse.2, reverse.1, cat.1a, IH

Conclusion: By the induction principle for sequences, $P(s)$ holds for all finite sequences $s$. Therefore:

$$\forall\, s \;\bullet\; \text{reverse}(\text{reverse}\; s) = s$$

The function reverse is an involution: applying it twice returns the original sequence.
